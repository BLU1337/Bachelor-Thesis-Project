Open\+CV 3 with contrib modules needs to be installed via CMake. Add a environment variable \char`\"{}\+Open\+CV\+\_\+\+DIR\char`\"{} to the path of the corresponding install root directory. Then build the project itself as usual.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{Camera Calibration}\label{md__r_e_a_d_m_e_autotoc_md2}
To calibrate Camera use pattern.\+png on a piece of cardboard. To start Calibration add \char`\"{}-\/c\char`\"{} Commandline Option. To Calibrate you need at least 15 valid pictures of Chessboard pattern. To save a picture while in calibration mode press Space. After at least 15 valid Pictures (around 50 are recommended) press Enter to start calibration Process.\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{Marker Generation}\label{md__r_e_a_d_m_e_autotoc_md3}
